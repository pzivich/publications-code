## File Manifesto

`data/`
- `actg.csv`: AIDS Clinical Trial Group data set used for all the case studies
- `actg_predict_g50.csv`: data set for generating predictions in Case 3 with 50 grid values
- `actg_predict_g1000.csv`: data set for generating predictions in Case 3 with 1000 grid values

`Case-Studies.ipynb`
- Python JuPyter notebook walking through the three case studies in the main paper

`Case-Studies.R`
- R code walking through the three case studies in the main paper

`Case-Studies.sas`
- SAS code walking through the three case studies in the main paper

`Case-Studies-Appendix.ipynb`
- Python JuPyter notebook walking through the three case studies in Appendix 2
